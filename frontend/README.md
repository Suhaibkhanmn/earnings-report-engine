# Earnings Call Intelligence Engine - Frontend

Minimal, classy light-mode interface for viewing earnings call reports.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8001
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Features

- Clean, minimal design optimized for finance professionals
- Dropdown selection for ticker and quarters
- Expandable sections for report viewing
- Evidence-backed claims with citations
- Comparison view (current vs previous quarter)
